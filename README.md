# TD-东哥 Image Compress Ultra

Chrome / Edge 扩展：在网页选择图片上传前自动拦截，智能压缩后再交给网站。

## 功能

- 捕获阶段拦截 `<input type="file">` 的图片选择
- **拖拽上传拦截**（全页 drop 捕获 + 提示遮罩）
- **压缩前后缩略图**对比，逐张显示体积变化
- **单张跳过**：可对某张保留原图，其余仍压缩
- 智能输出：**WebP / JPEG 择优**（可选手动指定）
- 透明图可保留 PNG 候选；压缩后几乎不变小则**保留原图**
- 最大边长等比缩放（默认 1920px，同时限制宽高）
- 小于阈值的图片不拦截（默认 100KB）
- 多选时非图片文件原样保留
- 全部本地 Canvas 处理，不上传服务器
- Esc / × 关闭浮层
- 扩展图标（16/32/48/128）

## 安装（开发者模式）

1. 打开 `chrome://extensions`（Edge：`edge://extensions`）
2. 开启「开发者模式」
3. 「加载已解压的扩展程序」→ 选择本仓库目录
4. 刷新目标网页后，选择图片上传即可看到浮层

## 设置

点击扩展图标：

| 项 | 默认 | 说明 |
|----|------|------|
| 自动检测 | 开 | 关闭后完全不拦截 |
| 压缩质量 | 80% | JPEG/WebP 质量 |
| 最大边长 | 1920px | 超过后等比缩小 |
| 最小触发 | 0KB | 小于此值不弹层（0=全部拦截） |
| 调试日志 | 关 | 控制台输出排查日志 |
| 输出格式 | 智能 | auto / webp / jpeg |
